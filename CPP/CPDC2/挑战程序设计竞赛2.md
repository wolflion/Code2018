    + `findMaximum(A,l,r)  m=(l+r)/2   if l==r-1//只有一个元素  return A[1]  else u = findMaximum(A,l,m)//递归前半部分  v = findMaximum(A,m,r)//递归后半部分  x=max(u,v)//Conquer  return x`
